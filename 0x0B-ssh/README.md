# Project: 0x0B-ssh

## Learning Objectives
+ What is a server
+ Where servers usually live
+ What is SSH
+ How to create an SSH RSA key pair
+ How to connect to a remote host using an SSH RSA key pair
+ The advantage of using #!/usr/bin/env bash instead of /bin/bash
